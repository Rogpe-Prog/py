## Itens da lista

A atribuição é possível para partes de listas, assim como para elementos individuais de uma lista.
Dessa forma, você pode até mesmo alterar o tamanho de uma lista ou limpá-la completamente fazendo algo como:

```python
animais[:] = []
```
  
### Tarefa
Torne todos os `animais` elefantes, substituindo os últimos dois itens.
Sugerimos que você execute o programa na IDE sem fazer nenhuma alteração. Isso permitirá que você veja como a lista `animais` se parece depois que todas as operações foram realizadas.

<div class='hint'>Use a atribuição para uma parte, como nos exemplos.</div>

<div class='hint'>Você pode atribuir uma lista consistindo de vários elementos.</div>
